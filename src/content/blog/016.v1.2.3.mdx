This update focuses on fixing issues and improving overall usability of the Calendar.

## Improvements

- The Calendar is now fully keyboard-navigable, allowing users to move through months and select dates using the keyboard.
- Improved accessibility for screen readers, with better descriptions and navigation cues for calendar elements.
- Improved performance when rendering the calendar, especially for calendars with a large number of events.
- Updated the event editing interface, making it more intuitive and easier to use.

## Bug Fixes

- Fixed the reset password button when editing a user being too small for its container.
- Fixed issues with the 2FA enrollment flow.
