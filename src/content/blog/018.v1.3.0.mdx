This update focuses on improving account management, streamlining administration, and enhancing the experience on larger displays. It also introduces integration with the new Account App and includes several usability improvements and bug fixes.

## Improvements

- The **Admin** page has been reorganized for improved usability and navigation.
- You can now add multiple email addresses to your account and choose a primary email address for notifications.
- The **Announcements** page now supports real-time updates and pagination for faster load times.
- Overall application performance has been improved, resulting in significantly faster page loads.
- The **Standings** page has been redesigned for large screens with improved spacing and readability.
- Integration with the new **Account App** has been added. Learn more in the announcement [here](https://origamisystems.org/blog/new-account-app).
- Editing users and houses has been simplified with a single **Save** button instead of saving each field individually.
- The **Point Presets** page has been redesigned with a new table layout for a cleaner and more efficient editing experience.

## Hiding Points During a Game

To help build suspense or prevent the app from revealing the winner too early, you can now hide point totals while a game is in progress.

This option is available from the **Add Points** page by enabling the **Hide Points** toggle. When enabled, point totals will be hidden from the **Standings** page. Points can be made visible again at any time.

## Bug Fixes

- Fixed an issue where approving points in the Admin panel did not clearly indicate what was being approved.
- Fixed an issue where some tables did not display all available fields.

## Bugs and Feedback

As always, if you encounter any bugs or have suggestions for improving The House App, we'd love to hear from you. You can submit feedback from the **Account** page.
